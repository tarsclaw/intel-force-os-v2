
- `hh_decision_trigger` at start of every run
- `hh_decision_output` before producing the artefact
- `hh_decision_action` when the human resolves

These are sourced from `${CTX_AGENT_DIR}/.claude/hooks/_shared/hook-helpers.sh`. If `_shared/` is not present, fail the session — do not proceed silently.
